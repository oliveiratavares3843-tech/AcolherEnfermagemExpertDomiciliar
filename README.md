# Sistema de Gestão de Enfermagem Expert Domiciliar

Este é um sistema especializado para o controle e gestão de atendimentos de enfermagem domiciliar, desenvolvido para oferecer precisão e agilidade no acompanhamento de pacientes.

## 🚀 Tecnologias Utilizadas
- **Python**: Linguagem principal.
- **Streamlit**: Interface do usuário e dashboard.
- **Pandas**: Manipulação e organização de dados.
- **SQLite**: Armazenamento seguro de informações.
